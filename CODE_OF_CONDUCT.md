# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be decent. Assume good faith. Disagree about the work, not the person.

Literary geography is full of genuine, unresolvable disagreement — where exactly
Clarissa lived, which building Dostoevsky meant by "S— Place". That disagreement
is a feature of this project, not a problem with it. Argue the evidence, mark it
`disputed`, and move on.

Report problems to the maintainers via a private issue or email.
